# Premium Frontend

Esta carpeta agrupa la experiencia Premium del frontend.

Responsabilidades permitidas:

- Componentes reutilizables para estado de licencia.
- Indicadores Community/Premium.
- Mensajes de grace period, vencimiento, revocacion o licencia invalida.
- Bloqueos visuales para features Premium.
- Resumen de limites y uso.

Responsabilidades prohibidas:

- Guardar secretos.
- Mostrar tokens de activacion completos.
- Forzar Premium desde el cliente.
- Implementar logica de firma o verificacion criptografica.

La pantalla principal de licencia sigue integrada en `Configuracion > Licencia`.
