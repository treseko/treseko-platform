# Politica De Marca

El codigo de Treseko Community se publica bajo AGPL-3.0-or-later, pero la marca no se licencia bajo AGPL.

## Puedes

- mencionar Treseko para describir compatibilidad, forks o integraciones;
- enlazar al sitio oficial [treseko.com](https://treseko.com);
- indicar claramente que tu version es una modificacion o fork no oficial.

## No puedes

- presentarte como Treseko oficial sin autorizacion;
- usar el nombre, logo o identidad visual para confundir usuarios;
- publicar servicios, binarios o builds modificados como si fueran releases oficiales;
- registrar dominios, paquetes o cuentas que aparenten ser canales oficiales de Treseko.

## Forks

Si distribuyes una version modificada, usa un nombre distinguible y aclara que no es un release oficial de Treseko.

## Contacto

Para autorizaciones de marca, partnership o uso comercial, usa los canales publicados en [treseko.com](https://treseko.com).
